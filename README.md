# Portfolio
Перед вами учебный лабораторный стенд для изучения базовых механизмов построения и администрирования компьютерных сетей, созданный и реализованный мной.
В рамках стенда были протестированы основные сетевые технологии: VLAN, межсетевой маршрутизация, DHCP, NAT, статическая и динамическая маршрутизация.Все проекты выполнены в рамках обучения по направлению "Информационная безопасность". Также, помимо лабораторного стенда прикреплен сертификат о прохождении курсов по сетевым технологиям, 2-й части. К сожалению сертификат первой части был утерян.

# Используемые технологии
- TCP/IP
- VLAN
- Router-on-a-Stick
- DHCP
- NAT (Source NAT, Destination NAT)
- Static Routing
- RIP
- OSPF
- Wireshark

# Результаты

В результате выполнения лабораторного стенда были получены практические навыки:

- сегментация сети с использованием VLAN
- организация межсетевой маршрутизации
- автоматическая выдача IP-адресов по DHCP
- преобразование сетевых адресов NAT
- настройка статической и динамической маршрутизации
- анализ сетевого трафика с помощью Wireshark
